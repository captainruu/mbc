🏋️ Muscle Beach Club — Website
Premium gym website for Muscle Beach Club, Uluwatu, Bali.
🚀 Deploy to GitHub Pages

Create a new GitHub repository (e.g. muscle-beach-club)
Upload the index.html file to the root of the repository
Go to Settings → Pages
Under Source, select main branch → / (root) → Save
Your site will be live at: https://yourusername.github.io/muscle-beach-club/

✅ Features
Landing Page

Multi-language (🇬🇧 English / 🇮🇩 Indonesian)
Auto-sliding hero carousel (5 images)
About section with stats
Facilities grid with photos
Membership packages with register buttons
Additional services section
Contact with WhatsApp, Instagram & Google Maps buttons
Google Maps embed

Promo Page

Dynamic promo/event cards
Managed via Admin dashboard

Membership Registration

Pop-up form for each package
Redirects to WhatsApp admin for processing

Member Dashboard

Login via unique Member ID (no password)
Shows package, start date, expiry date
Active/Expired/Expiring status badge
Progress bar for membership period
Request extension → redirects to WhatsApp
Log out button

Admin Dashboard

Login with #ADMIN0#
Data Member tab: searchable/filterable member table, edit & delete
Add Member tab: full registration form with multi-level discount
Auto-generated unique Member IDs (MBC-XXXX)
Edit Packages tab: change all membership prices + global promo discount
Promo/Events tab: add and remove promotional banners
Download Data: export filtered member list as CSV
Expired-today notification banner

AI Chatbox

Built-in keyword-based AI assistant
Answers questions about facilities, packages, location, contact


🛠 Tech Stack

Pure HTML5 + CSS3 + Vanilla JavaScript
No frameworks, no build tools, no dependencies
Data persisted via localStorage
Google Fonts (Barlow Condensed + Barlow)
